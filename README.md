# NaNLABS

This repository contains the organization-level configuration for [NaNLABS](https://nanlabs.com) on GitHub:

- **Profile README** (`profile/README.md`) — shown on our [organization page](https://github.com/nanlabs)
- **Pull Request template** (`.github/PULL_REQUEST_TEMPLATE.md`) — applied to all PRs across organization
- **Issue templates** (`.github/ISSUE_TEMPLATE/`) — bug reports, feature requests, documentation
- **Code owners** (`.github/CODEOWNERS`) — global ownership rules
- **Workflows** (`.github/workflows/`) — CI/CD for org-wide tooling (Danger, etc.)
- **Contributing guide** (`CONTRIBUTING.md`) — contribution guidelines for open-source projects

For the public organization profile with all open-source projects, see [`profile/README.md`](profile/README.md).
